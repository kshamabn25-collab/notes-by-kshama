Change Impact Analysis

Change trigger

A key milestone forecast moves due to updated inputs (approvals, procurement, readiness). This analysis captures downstream impact, dependency chain effects, and decision options.

Required inputs

  • Original baseline milestone date + rationale
  • Updated forecast date + evidence
  • Impacted downstream milestones (by dependency chain)
  • Mitigation options + recommended path

Decision output (what leadership gets)

  • Impact-to-date summary (what changed and why)
  • Committed-date exposure (risk and confidence)
  • Tradeoffs (cost, risk, sequence, resource moves)
