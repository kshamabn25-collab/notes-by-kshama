Integrated Master Schedule (IMS)

WBS structure (overview)

  • Level 1: Global Capacity Portfolio
  • Level 2: Regional Programs (EMEA / NAM / APAC)
  • Level 3: Site phases (Civil, Power, Network, Commissioning, Handover)
  • Level 4: Discipline activities with owners and dependencies

Milestone chain (end-to-end)

  • Site ready → Civil complete → Power energised → Network integrated → Commissioning complete → Operations handover

Integration points (examples)

  • Power energisation gates network commissioning readiness
  • Shared procurement lead times influence multiple regional forecasts
  • Operations handover depends on cross-site completion + readiness sign-offs

How this is used

  • Weekly refresh of critical/near-critical drivers
  • Dependency breaches escalated via Dependency Register
  • Changes to committed dates governed via Change Impact Analysis
