Critical Path Analysis (CPM)

Critical path drivers (portfolio)

  • Utility approvals → Power energisation → Commissioning readiness → Handover
  • Long-lead equipment delivery → Install → Test → Commission
  • Network integration window alignment across regions

Near-critical monitoring

  • Near-critical band: ≤ 10 working days float
  • Float erosion flagged even if milestone dates are unchanged
  • Dependencies without buffer require mitigation planning

Interpretation rules (used in reviews)

  • Green: float stable, forecast aligned, no dependency breaches
  • Amber: float erosion or forecast volatility, mitigation required
  • Red: committed date risk, recovery plan + executive decision required
